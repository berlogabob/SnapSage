# SnapSage

**Все твои скриншоты — умная поисковая база знаний.**

Делаешь скриншот → через несколько секунд он уже с текстом, описанием, переводами, тегами и автоматически добавлен в календарь и Logseq.

![SnapSage Demo](https://via.placeholder.com/800x400?text=SnapSage+Demo)

---

### ✨ Основные возможности

- Автоматический захват скриншотов из любых источников:
  - Яндекс Диск (включая папку «Снимки экрана»)
  - Google Drive, iCloud, Рабочий стол, Downloads, галерея телефона и любые папки
- Точный OCR + **визуальное понимание** (qwen3.5:2b + glm-ocr)
- Перевод на выбранные языки (русский, английский, португальский и др.)
- Автоматическое определение событий и добавление в Google Calendar
- Глубокая интеграция с заметками:
  - Logseq (Daily Notes + linked blocks)
  - Obsidian, Notion, Bear
- Умные теги + ручная правка
- Мощный поиск по эмбеддингам и всем языкам
- Полностью оффлайн (Ollama локально)
- Работает на **всех платформах**: Android, iOS, macOS, Windows, Linux (.AppImage)

---

### 🚀 Быстрый старт

1. Скачай последнюю версию:
   - Android → `.apk`
   - iOS → `.ipa`
   - macOS → `.dmg`
   - Windows → `.exe`
   - Linux → `.AppImage` (сделай исполняемым)

2. Запусти приложение
3. Пройди простой онбординг (3 минуты)
4. Выбери папки/диски и языки
5. Готово! Просто делай скриншоты как обычно.

---

### 🛠 Технологии

- **Flutter** 3.29+ (один код для всех платформ)
- **Ollama** + `qwen3.5:2b` (основная) и `glm-ocr` (таблицы)
- **Drift** (SQLite)
- **Riverpod** 2.5
- Logseq Graph API, Obsidian Local REST, Notion API
- 100 % оффлайн по умолчанию
